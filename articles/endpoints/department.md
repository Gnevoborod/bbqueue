## Получение иерархического списка отделов\целей

**Адрес**

/targets_hierarchy

**Метод**
GetHierahchy

**Тип эндпоинта**

GET

**Цель**

Эндпоинт возвращает данные для отображения на фронте (разделы, подразделы и цели в иерархии)

**Входные параметры**

*Отсутствуют*

**Алгоритм**

1. Вычитываются данные из таблиц group, target.
2. Формируется иерархия по логике, описанной в /mechanics/target_hierarchy.md
2. Формируется json с иерархическим представлением списка отделов\целей, для отображения на фронте.
Примечание: По сути на основании этого json будет формироваться "путь" на экране выдачи талона по целям. 
Например: Обслуживание физ.лиц->Кредитование->Кредитные карты
