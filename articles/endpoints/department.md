#Эндпоинт с API отделовцелей#

##Получение иерархического списка отделов\целей##
**Тип эндпоинта**
GET

**Входные параметры**

*Отсутствуют*

**Алгоритм**

1. Формируется json с иерархическим представлением списка отделов\целей, для отображения на фронте.
Примечание: По сути на основании этого json будет формироваться "путь" на экране выдачи талона по целям. 
Например: Обслуживание физ.лиц->Кредитование->Кредитные карты

**Выходные параметры**

Будет спроектирован отдельно в специальном файле.
