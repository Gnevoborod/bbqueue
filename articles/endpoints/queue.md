# Эндпоинт с API талонов

## Отображение очереди

**Адрес**

/queue

**Тип эндпоинта**

GET

**Метод**

GetAllTicketsFromQueue

**Входные параметры**

Без параметров

**Алгоритм**

Возвращает список всех талонов в состоянии Обрабатывается.

**Выходные параметры**

|Параметр|Тип|Обязательность|Комментарий|
|--|--|--|--|
|TicketNumber|string|yes|Номер талона|
|WindowNumber|string|yes|Номер окна|

## Прослушивание очереди на предмет наличия новых посетителей по конкретной цели

**Адрес**

/customer_appear/{TargetCode}

**Тип эндпоинта**

GET

**Метод**

ListenForCustomerCame

**Входные параметры**

|Параметр|Тип|Обязательность|Комментарий|
|--|--|--|--|
|TargetCode|string|yes|Номер цели|

**Алгоритм**

Возвращает информацию о появлении посетителя по конкретной цели в очереди.

**Выходные параметры**

|Параметр|Тип|Обязательность|Комментарий|
|--|--|--|--|
|CustomerCame|bool|yes|Флаг о прибытии посетителя|
